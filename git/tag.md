```bash
git tag
```
> 태그 목록 보기<br>

```bash
git log --decorate
```
> log 명령어에 --decorate 옵션을 붙여 실행하면, 태그 정보를 포함한 이력을 확인할 수 있습니다.<br>


```bash
git tag -am
```
> 주석 달린 태그 추가하기<br>

```bash
git tag -n
```
> -n 옵션을 지정하여 tag 명령어를 실행하면 태그 목록과 주석 내용을 확인할 수 있습니다.<br>

```bash
git tag -d [tagname]
```
