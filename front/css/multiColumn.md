
# multiColumn(다단 디자인)
멀티 칼럼 레이아풋 모듈은 이전 CSS 방법 보다 유연성을 제공합니다.<br>

## 칼럼 개수 또는 폭 설정
### column-count
개수 설정<br>
### column-width
폭 설정<br>
### columns
속기형<br>

## 칼럼 간격 
### column-grap
간격 설정
### column-rule
스타일 지정
> column-rule-color<br>
> column-rule-style<br>
> column-rule-width<br>

## 칼럼 병합
### column-span
테이블 처럼 병합<br>

## 칼럼 채우기
### column-fill

## 브레이크
### column-after: column
